Configure Windows Defender

One thing to mention at the top, for Windows human users,
If Windows Defender isn't holding too much system resources,
keep Defender running is the best solution.

You must achieve the abilities to protect yourself from chaos,
otherwise, install a third-party anti-virus software.
I left Windows Defender running on my workstation,
but turn Defender off on most virtual machines.

Make this decision beware.

To turn off Windows Defender in 1909, you need to turn off Tamper Protection first,
then use group policy to actually turn off Windows Defender. It can't be turn off by group policy directly.
