Configure Windows Search

You can skip this part if you're relying on this feature.
Yes, Windows Search is awesome on some pro's hand, those secret features...
but the more only use it on startup menu to find specify program's shortcuts.
By default, Windows Search is heavy, a bit slow and do use less real-time internet search at the same time(Version 1803).
The people don't rely on this feature will feel bad when they just want to open Photoshop,
but Windows takes five seconds to search a simple "lnk"(Windows shortcuts file extension name).
And Cortana displays a Bing search result about Photoshop.
Lots of system resource holds in this workflow, but most import thing is,
it's not fast enough.

Indexing Options

To make Windows Search(Startup menu search) become a program launcher you ever meet in Phone,
execute the following steps.
  1. Open Control Panel
  2. Open Indexing Options
  3. Modify
  4. remove all checkbox item and left only "Start Menu"
  5. Click on OK button to save changes, suddenly Windows begins to remove entries other than Start menu
Close Indexing Options, soon Windows Search will have faster response.
This setting makes Windows Search feature only cache search Start Menu shortcuts now.
My suggestion "Remove other location" may not cover your working files, so, apply this setting or not by your choice.

After 1803 update, group policy doesn't work to block web search.(Pro Version)
