Configure Windows Search

You can skip this part if you're rely on this feature.
Yes, Windows Search is awesome on some pro's hand, lots of features...
but the more only use it on startup menu to find specify program's shortcuts.
By default, Windows Search is heavy, a bit slow and do use less realtime internet search at the same time.
The people doesn't rely on this feature will feel bad when they just want to open Photoshop,
but Windows use five seconds to search a simple lnk(Windows shortcuts file extension name).
And cortana display a Bing search result about Photoshop.
Lots of system resource holds in this workflow, but most import thing is,
it's not fast enough.

To make Windows Search(Startup menu search) become a program launcher you meet in Phone,
Open Control Panel , find "Indexing Options", click on it.

Indexing Options

Click on the Modify button on Indexing Options dialog,
remove all check box item and left only "Start Menu".
Click on OK button to save changes, suddenly Windows begins to remove entries other than Start menu.
Close Indexing Options, soon Windows Search will have fast response.
This settings removed Windows Search features and it can only search Start Menu shortcuts now.
Apply this settings or not, by your choice.
